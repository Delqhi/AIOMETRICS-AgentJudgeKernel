# Judge Kernel Context

## Problem Statement
Agents can diverge in architecture, security, and scope when no deterministic governance source is enforced.

## Context Goal
Provide one operational kernel notebook that defines rules, constraints, contracts, and missing-work detection.

## Mandatory Query Types
- Constraint query: active `critical_invariant` for module.
- Gap query: missing artifacts until Definition of Done.
- Escalation query: conflict handling path for unresolved contradictions.
